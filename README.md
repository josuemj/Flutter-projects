# Flutter Projects

This repository contains various Flutter projects.

## How to Run a Project

Navigate to the specific project directory you wish to run (e.g., `cd buttons`).

### macOS

To run a Flutter project on the default iOS Simulator:

1.  Open the iOS Simulator:
    ```bash
    open -a Simulator
    ```
2.  Once the simulator is open, run the Flutter project:
    ```bash
    flutter run
    ```

### Windows

To run a Flutter project on a connected Android device or emulator:

1.  Ensure you have an Android device connected or an Android emulator running.
2.  Run the Flutter project:
    ```bash
    flutter run
    ```

To run a Flutter project as a Windows desktop application (if enabled for the project):

1.  Run the Flutter project targeting Windows:
    ```bash
    flutter run -d windows
    ```
